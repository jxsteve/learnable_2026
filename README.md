## What is Version Control?
Version control is a system that tracks changes to files over time. It allows developers to see what changes were made, who made them, and when they were made. It also helps teams collaborate without overwriting each other's work.

## Difference Between Git and GitHub
Git is a version control system used to track changes in code locally on your computer.

GitHub is a cloud-based platform that hosts Git repositories and allows developers to collaborate, share code, and manage projects online.

In simple terms:
- Git = tool
- GitHub = platform

## Difference Between git fetch and git pull
`git fetch` retrieves updates from a remote repository but does not apply them to your current branch.

`git pull` retrieves updates and automatically merges them into your current branch.

In simple terms:
- fetch = download changes only  
- pull = download + apply changes
